Trabajando con Java

La lectura por teclado en java puede hacerse de distintas formas, una de ellas es implementando el paquete java.utils.Scanner, posteriormente deberemos crear un objeto de la clase Scanner de esta forma Scanner lector=new Scanner(System.in) hecho esto, ya podremos leer valores por teclado. Esto puede servirnos por ejemplo para asignar valores a variables, por ejemplo, pongamos que tenemos una variable num de tipo int, para asignarle un valor llamariamos al objeto que acabamos de crear de esta forma num=lector.nextInt(). Ya en la consola de comandos deberemos introducir un valor adecuado y preionar la tecla enter sin embargo debemos de tener cuidado con no introducir valores incompatibles, si la variable es un int y nosotros introducimos un float, el programa finalizara y tirara un error tal y como "input mismatch exception"

Ejemplo de codigo: 				
					
import java.util.Scanner;
public class E1{
	public static void main(String[] args){
		Scanner lector=new Scanner(System.in);
		String nombre;
		System.out.println("Introduce tu nombre");
		nombre=lector.nextLine();
		System.out.println("Hola "+nombre);
	}
}
